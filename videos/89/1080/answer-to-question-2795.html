Answers:

You can use imread command to load the file in an array(1), imshow to display the image(2), and shape to view the dimensions of the array(3). Array slicing is used to get the parts of the image(4)
For example 
1. a=imread("filename")
2. imshow(a)
3. a.shape
4. a[255:, :255]
