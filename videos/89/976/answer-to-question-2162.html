Answers:

try this code:

nes_lst = [1, 2, 3, ['a', 'b', 'c'], 7, 8, 'b']

for element in nes_lst:
    if isinstance(element, list):
        if 'b' in element:
            element.remove('b')
   else:
        if element == 'b':
            nes_lst.remove('b')
