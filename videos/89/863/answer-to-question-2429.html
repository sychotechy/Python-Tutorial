Answers:

You can save the code that you have typed in the IPython session using the %save magic command

Link: https://ipython.readthedocs.io/en/stable/interactive/magics.html#magic-save

In case you are looking for the downloadable videos of the sessions/videos please refer this doc https://process.spoken-tutorial.org/images/1/1b/Download-Tutorials.pdf
